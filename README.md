# Gestión de flotas de drones

Proyecto React + Vite preparado para desplegar en GitHub Pages.

## Antes de subirlo
Edita `vite.config.ts` y cambia:

```ts
base: "/NOMBRE_DEL_REPO/",
```

por el nombre real de tu repositorio.

## Despliegue
1. Sube el proyecto a GitHub.
2. Haz push a la rama `main`.
3. En GitHub ve a **Settings > Pages**.
4. Selecciona **Source: GitHub Actions**.

## Desarrollo local
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
